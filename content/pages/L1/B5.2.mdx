### Learning aim
By the end of this page, the learner should be able to give and interpret four-figure grid references using printed map extracts or training grids.

### What a four-figure grid reference does
A four-figure grid reference identifies one whole grid square on the map. It does not pinpoint the exact feature inside the square, but it tells you which square to search in.

This is often enough for simple beginner exercises and is the foundation for more precise work later.

### How to read it
To give a four-figure grid reference:

1. Find the easting of the left-hand grid line of the square.
2. Find the northing of the bottom grid line of the square.
3. Put them together in that order.

### Worked example
A bridge lies in the square:

- right of easting 34
- above northing 62

Its four-figure grid reference is **3462**.
