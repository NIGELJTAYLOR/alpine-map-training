### Learning aim
By the end of this page, the learner should be able to locate and name the main parts of a topographic map: title, legend/key, scale bar, north arrow and grid lines.

### The main parts of a map
Most outdoor maps share a common set of structural features:

- **Title and series name** – tells you which area the map covers and which map family it belongs to.
- **Scale bar and scale statement** – shows the relationship between map distance and ground distance (for example 1:25,000).
- **Legend or key** – explains the colours and symbols used on the map.
- **North arrow** – shows which direction on the map is north.
- **Grid lines and grid numbers** – divide the map into squares so you can pinpoint locations.

Being able to find these parts quickly is essential. Before using any new map, you should spend a few minutes finding and checking each one.

### Worked example
Imagine opening a new 1:25,000 mountain map. Before you do anything else:

1. Find the **title** to confirm you have the right area.
2. Find the **scale statement and scale bar** to check the scale and contour interval.
3. Find the **legend** and scan for key symbols (paths, huts, cliffs, lifts, forests).
4. Find the **north arrow** so you know which way is north on the sheet.
5. Look for **grid lines and grid numbers** around the edges.

This habit means you always know what the map is showing and how to read it.
