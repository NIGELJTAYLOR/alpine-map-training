### Learning aim
By the end of this page, the learner should understand why rotating the map helps reduce confusion and should be able to turn a printed extract so that it better matches the world around them.

### Why rotate the map?
A map is easiest to use when the directions on the paper match the directions of the terrain around you. If the map is held at an awkward angle, the learner has to keep translating left/right and forward/backward in their head.

Rotating the map helps reduce that mental effort.

### Key idea
You are not changing the map itself. You are changing **how you hold it** so that the features on the paper point the same way as the features on the ground.

### Worked example
Imagine you are standing on a path junction. On the map, the path you want runs away to the north-east. If, in real life, that path heads away in front-right of where you are standing, turning the map so that the path points the same way can make the whole picture easier to understand.
