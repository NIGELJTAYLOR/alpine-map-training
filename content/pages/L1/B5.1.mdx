### Learning aim
By the end of this page, the learner should be able to identify eastings and northings on a map grid and explain the order in which they are read.

### The basic idea
Grid lines turn the map into a set of squares. These squares help the navigator describe location clearly.

There are two directions to remember:

- **Eastings** – the vertical grid lines, read from left to right.
- **Northings** – the horizontal grid lines, read from bottom to top.

A simple memory phrase is:

**Along the corridor, then up the stairs.**

That means:

1. Read the easting first.
2. Read the northing second.

### Worked example
Imagine a symbol sits inside the square between easting 23 and 24, and between northing 57 and 58.

To identify the square:

- read the left-hand grid line first → 23
- then read the bottom grid line → 57

So the square is **2357**.
