If a learner confidently calls every V shape a valley regardless of contour direction, that is the single most important error to correct. Return them to Exercise 1 and have them annotate the contour values on the page before naming each feature.

If a learner sketches valleys and spurs as the same shape, work through the worked example again on the board. Drawing the inverse pairing on the same diagram is more effective than re-explaining either alone.

If a learner skips Exercise 4 because "I can't picture it", that is the exact developmental need the workbook is built around. Coach them through the imagine-describe-sketch sequence used in C2.1 rather than letting them substitute a written description for the visualisation.

For the IMS / EMS connection: at assessment, the candidate is expected to read these features fluently from the map, give the aspect of each side of a crest in compass terms, and use the result in route selection. The exercises here are pitched at that fluency, so mark for both correctness and speed of recognition.
