*Note for trainer: model answers are illustrative. Where exercises depend on the Section C2.1 map extract, use the trainer reference below if you are using the standard extract; otherwise mark for the underlying understanding rather than the specific figures.*

### Exercise 1 — Identify the summit

Mark by checking that the learner has:

- recognised that a closed-loop pattern with the highest contour at the centre indicates a summit,
- and given a one-line reason that refers to the loop pattern, the contour pattern, or the position of the spot height.

For trainer reference, the standard Section C2.1 extract is configured as:

| Point | Answer | Model reason |
|---|---|---|
| A | Summit | Closed contour loops with the highest value at the centre and a spot height marking the top. |
| B | Not a summit | Contours run across the map without forming a closed loop; this is a slope, not a high point. |
| C | Summit | Smaller closed-loop pattern with rising height towards the centre; identifiable as a secondary peak. |

### Exercise 2 — Read the height

For the standard extract (summit at point A):

1. Highest contour value: **2,650 m** (accept any reading consistent with the trainer's print of the extract).
2. Spot height: **2,658 m** (accept any reading consistent with the trainer's print of the extract).
3. The spot height is more precise. The centre contour shows the highest contour line drawn on the map, but the actual top sits between two contours. The spot height marks the actual highest point with metre-level precision.

### Exercise 3 — Match the summit shape

For the standard extract:

1. Pattern D: **sharp peak**
2. Pattern E: **rounded summit**
3. Pattern F: **plateau-like high**

Mark by checking the learner has used the spacing of the loops as the cue, not the absolute height. A 2,200 m rounded summit and a 4,000 m sharp peak both follow the pattern rules.

### Exercise 4 — Side profile sketch

A strong sketch should:

- show ground rising to a single highest point,
- match the chosen summit type:
  - **sharp peak:** a narrow, pointed top with steep flanks,
  - **rounded summit:** a broad, gently curved top,
  - **plateau-like high:** a flat top with a small bump or spot height marking the actual peak,
- and label the highest point.

The sketch does not need to be artistic. It needs to show that the learner has translated the contour pattern into a 3D mental picture.

### Exercise 5 — "If I stood here"

A strong answer should include:

- a description of the ground immediately around the learner that matches the summit shape (flat for plateau, sloping away on all sides for a sharp peak, broad gentle slope for a rounded summit),
- a clear direction of any obvious drop, expressed in compass terms or as a feature direction (for example "the ground falls away steeply to the south-east"),
- and at least one neighbouring high point or recognisable feature.

Coach learners who slip into describing the symbol rather than the imagined view. The point of this exercise is to build the 2D-to-3D translation habit that Level 4 IMS will rely on.
