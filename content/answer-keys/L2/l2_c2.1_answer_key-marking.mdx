If a learner struggles to identify summits, return to the closed-loop / water-running-downhill mental model from C1.1 and C1.2. The conceptual step from "lines of equal height" to "closed loop equals high point" is small but easy to miss.

If a learner reads only the contour value and ignores the spot height, treat that as a notable error to correct. On the IMS log book and route cards, learners are expected to record spot height altitudes for waypoints, not contour values.
