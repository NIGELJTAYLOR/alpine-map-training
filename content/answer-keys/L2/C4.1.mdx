### Exercise 1 — Mark the high and low points

Mark by checking that the learner has read the contour values directly from the map rather than guessing. The exact values depend on the standard extract.

For trainer reference, the standard Section C4.1 extract is configured as:

| Slope | Highest contour (m) | Lowest contour (m) |
|---|---|---|
| A | 2,500 | 2,440 |
| B | 2,720 | 2,640 |
| C | 2,310 | 2,180 (range; slope curves and lowest point varies) |
| D | 1,950 | 1,860 |

If the learner reads only the values without identifying which is higher and which is lower, that is the marking point to coach: knowing the values is not the same as knowing the direction of slope.

### Exercise 2 — Draw the fall-line arrow

A correct fall-line arrow:

- runs perpendicular to the contour at the chosen point,
- points from the higher contour towards the lower contour,
- and is positioned near the centre of the named slope, not at the edge.

Common errors to mark down:

- arrow runs parallel to the contour (often a sign the learner confused contour direction with downhill direction),
- arrow points uphill (sign that the contour values were not checked),
- arrow drawn on a curved slope at a single point rather than at the point of greatest representative steepness.

For the standard extract, the fall lines run:

| Slope | Fall-line direction (compass, approx.) |
|---|---|
| A | South-east |
| B | South |
| C | West (varies; see Exercise 4) |
| D | North-east |

### Exercise 3 — Explain how you know

A strong answer should include:

- a specific contour-value reading (which contour is at what height),
- the perpendicular-to-contour rule explicitly named,
- and a confirming feature (a stream symbol pointing towards the open end of the V, a spur with the V pointing downhill, a spot height at the upper end).

If a learner answers "because it looks downhill", coach into specificity. The IMS / EMS-relevant skill is the explicit reading; the visual intuition is a useful confirmation but not the diagnostic.

### Exercise 4 — Curving slope

For the standard extract, slope C is configured as a curving slope where the fall line rotates approximately 60 degrees across the slope width:

| Point on slope C | Fall-line direction |
|---|---|
| Left point | South-west |
| Centre point | West |
| Right point | North-west |

A strong one-sentence answer should explain that the contour lines themselves curve across the slope, so the perpendicular at each point also rotates, producing different fall-line directions at different points. If the learner says "because the slope is curved", coach into the explicit perpendicular-rotation reasoning.

This exercise sets up the aspect-on-a-spur reading at C4.2 Exercise 5: spurs and ridges are inherently curving features in the same way.
