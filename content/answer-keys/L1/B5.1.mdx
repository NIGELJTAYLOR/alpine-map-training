### Exercise 1 — Name the parts
1. Eastings are the vertical grid lines.
2. Northings are the horizontal grid lines.
3. Eastings are read from left to right.
4. Northings are read from bottom to top.

### Exercise 2 — Memory rule
Reflective. Accept any correct version of “along the corridor, then up the stairs” or equivalent.

### Exercise 3 — Square identification practice
- Example square = 4176
- 1. 1863
- 2. 5221
- 3. 0744

### Exercise 4 — Why order matters
A strong answer should explain that if the order is reversed, the reference points to a different square and may locate the wrong place entirely.
