If the learner struggles to find certain map parts, encourage them to:

- Always locate the title, scale, legend and north arrow as soon as they unfold a new map.  
- Practise with different map series so they see how layouts vary.  
- Use finger or pen-tracing to reinforce where these items live on each map.
