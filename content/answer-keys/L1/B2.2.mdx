### Exercise 1 — Timed anatomy drill

There are no fixed numeric answers here. Look for improvement between the first and second attempts, and check that:

- The learner can find each item without confusion.  
- The learner has correctly circled the items which took longest to find.

### Exercise 2 — Spot the differences

Answers will vary by map. Strong responses should include points like:

1. Legend location: correctly described for each map (for example, bottom-left on Map 1, back cover on Map 2).

2. How the scale is shown: both maps show a scale bar, for example, and one also shows a numeric statement like 1:25,000.

3. North arrow position: whether it is clearly visible and if its location differs between maps.

4. Which map is easier and why: for example, one has a clearer layout or more obvious legend placement.

### Exercise 3 — Quick-fire questions

1. Name three essential parts of a topographic map.  
   - Model ideas: Title, legend/key, scale bar, north arrow, grid lines.

2. Which map part tells you what symbols mean?  
   - Legend/key.

3. Which map part allows you to give precise positions using numbers?  
   - Grid lines and grid numbers.

4. Which map part allows you to convert centimetres on the map into ground distance?  
   - Scale bar (and scale statement).

### Exercise 4 — Personal checklist

This is reflective. Look for:

- Evidence that the learner has identified something they do well.  
- A realistic recognition of a weak area (for example, "I always forget where the legend is on foreign maps").  
- A specific habit they will adopt when picking up a new map.
