### Exercise 1 — Label a blank route card template

The eleven standard fields, in the order they appear on the IMS route-card template (Quick Reference Card 3), are:

1. **Leg number**
2. **Start point and altitude**
3. **End point and altitude**
4. **Compass bearing (magnetic)**
5. **Distance (map)**
6. **Estimated time**
7. **Altimeter trigger height**
8. **Key terrain features**
9. **Avalanche assessment**
10. **Decision point criteria**
11. **Escape route**

Mark each candidate label in order (Q1 through Q11). A candidate who labels nine or ten fields correctly is at working level. A candidate missing two or more has not internalised the full anatomy; recycle the field listing.

The most commonly missed fields in labelling exercises are **altimeter trigger height** (often confused with target altitude or end altitude; it can be either) and **avalanche assessment** (often labelled as "hazard notes" or similar).

### Exercise 2 - Function-matching

Correct matches:

| Field | Function letter |
|---|---|
| Leg number | d |
| Start point and altitude | a |
| End point and altitude | b |
| Compass bearing (magnetic) | g |
| Distance | f |
| Estimated time | h |
| Altimeter trigger height | k |
| Key terrain features | e |
| Avalanche assessment | i |
| Decision point criteria | j |
| Escape route | c |

Mark each match. The most common errors:

- **Swapping start and end points (a and b):** both refer to a feature with an altitude; the candidate may not have internalised which is the start and which is the end. Drill the convention.
- **Confusing key terrain features (e) with avalanche assessment (i):** both refer to features along the leg; the distinction is descriptive (terrain) vs assessable (avalanche). Coach the distinction.

### Exercise 3 - Fill in one leg

There is no fixed answer; the entry depends on the candidate's chosen leg from their exam-area map. Mark by checking that:

- all nine basic fields (excluding decision points and escape routes, which come in D9.3 and D9.4) are populated,
- the start and end altitudes are precisely known (per D5.2 reliable-known-points discipline),
- the bearing has been taken using the D2.1 procedure and converted using the D2.2 variation,
- the target altitude is the end altitude,
- the distance is measured at the map's scale,
- the estimated time is consistent with the candidate's typical ascent or descent rate,
- the key terrain features list multiple specific features (not just the obvious one),
- and the avalanche assessment uses the C1.2 / C4.2 / C6.2 vocabulary.

A candidate whose entry is missing fields or has vague entries should recycle D9.1 before moving to D9.2.

### Exercise 4 - Identify the typically under-described fields

A strong answer should:

- name **key terrain features** as one of the two under-described fields, with an example of an under-described entry (one obvious feature only) vs a complete entry (multiple features in order along the leg),
- name **avalanche assessment** as the other under-described field, with an example of an under-described entry (slope angle only) vs a complete entry (slope angle, aspect, runout, terrain traps, all referenced to the C6.2 vocabulary),
- and reference the candidate's own Exercise 3 entry for whether it shows the under-description pattern.

If the candidate names other fields as the typically under-described, that is acceptable as long as the explanation is sound. The key-terrain and avalanche fields are the most common in the trainer's experience but other fields can be problematic for specific candidates.
