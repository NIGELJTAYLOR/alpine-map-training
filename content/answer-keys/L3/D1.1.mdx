### Exercise 1 - Label the compass

A correct labelling identifies all six parts:

- baseplate (the transparent flat body),
- direction-of-travel arrow (the fixed arrow on the baseplate, pointing forward),
- rotating bezel (the dial marked 0 to 360),
- orienting arrow (the arrow on the bezel floor),
- magnetic needle (the floating needle, red end to magnetic north),
- parallel orienting lines (the lines on the bezel floor, parallel to the orienting arrow).

Mark by checking that each label points to the correct part. If a label is on the right part but uses the wrong name (for example "north arrow" instead of "orienting arrow"), accept the labelling but note the correct vocabulary.

### Exercise 2 - Match each part to its function

Correct matches:

| Part | Function letter |
|---|---|
| Baseplate | b |
| Direction-of-travel arrow | c |
| Rotating bezel | d |
| Orienting arrow | a |
| Magnetic needle | e |
| Parallel orienting lines | f |

Mark any wrong match. The most common error is swapping the orienting arrow (a) and the parallel orienting lines (f); the orienting arrow aligns with the magnetic needle in the field, while the orienting lines align with the grid lines on the map.

### Exercise 3 - The two most-confused parts

A strong answer should:

- name the direction-of-travel arrow and the orienting arrow as the two parts,
- explain that both are arrows on the same compass and both point,
- distinguish the function of each (direction-of-travel = where you are going; orienting = the alignment target for the needle),
- and state the consequence of confusing them (a bearing that is wrong, often by 90 degrees or more depending on the bezel position; in the worst case, a candidate walks the wrong direction).

If a learner gives a vague answer ("they look similar"), coach for the specific functional distinction.

### Exercise 4 - Quick check

For a compass with the bezel set to 90 and the magnetic needle sitting freely:

1. The direction-of-travel arrow points 90 degrees away from where the needle is currently sitting (because the bezel has rotated the orienting arrow 90 degrees from where the needle is, but the direction-of-travel arrow on the baseplate is fixed). In practical terms: the direction-of-travel arrow points where the user is currently facing; the needle still points to magnetic north.
2. To set the compass to 90 degrees and walk on it, **rotate the body** until the needle sits inside the orienting arrow on the bezel. The bezel has already been set to 90; rotating the body brings the needle into alignment.
3. To show a bearing of 270 degrees, the bezel must show 270 at the index mark (where the direction-of-travel arrow meets the bezel scale).

Mark by checking the candidate has not confused which thing rotates at which step. The bezel is set first (to the bearing), then the body rotates (to align the needle).
